# Pena Digital Homebrew Tap

Official Homebrew Tap for Pena Digital tools. Simplifies installation on macOS and Linux by automatically managing system dependencies.

## Install

```bash
brew tap pendig/tap
brew install rute-bayar
```

## Available Formulae

- `rute-bayar`: Payment router CLI and webhook daemon for Indonesian payment gateways.
- `videocut`: AI-powered video automation tool for content creators.
